# ReadMe

使用`python3`编写.

请安装`pycryptodome`依赖库.: 

`py -3 -m pip install pycryptodome`

`py shell.py` or `python3 shell.py`运行.

实现密码学要求`机密性`,`完整性`,`不可抵赖性`的模仿`https`的微型socket系统.

也算是作为自己对`pycryptodome`的熟悉.
